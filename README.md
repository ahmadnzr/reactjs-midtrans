## Get Started

- clone the repo
- run the backend service:
    - configure your environment use `.env-template`
    - run these commands :
        ```bash
        cd server/ && 
        yarn &&
        yarn devdev
        ```
- run the client
    - configure your environment use `.env-template`
    - run these commands:
        ```bash
        cd client/ &&
        yarn &&
        yarn start
        ```
    - open the browser and simulate the transaction.
